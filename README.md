# Cenk Mimarlık Website

A modern, multi-page construction company website built using React, designed to showcase construction and renovation projects in Bodrum.

---

## Live Demo

 https://cenk-mimarlik-website.vercel.app/

---

## Tech Stack

- React  
- Vite  
- React Router  
- JavaScript  
- CSS

---

## Features

- Multi-page navigation using React Router  
- Modern homepage with hero section  
- Services and project portfolio sections  
- Dedicated pages: About, Services, Projects, Contact  
- Reusable component-based architecture  
- Clean and minimal UI design
